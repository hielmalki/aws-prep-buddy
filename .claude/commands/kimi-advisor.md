Hole eine Zweitmeinung von Kimi K2 zu einer konkreten Frage oder einem Designproblem ein.

Schritte:
1. Nimm die Nutzerfrage sowie optional einen relevanten Code-Schnipsel oder Plan-Ausschnitt.
2. Rufe das Bridge-Skript auf:
   `echo "<Frage + Kontext>" | node .claude/bridges/kimi-bridge.mjs advisor`
3. Zeige Kimis Antwort 1:1 im Chat — kein File-Output, keine Paraphrase.
4. Lasse den Nutzer entscheiden, was mit der Antwort geschieht.

Hinweis: Für vollständige Implementierungspläne nutze /kimi-plan stattdessen.
